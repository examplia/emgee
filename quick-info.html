Quick Info

Stages + ControlPoints

Attack / Defend

  • Setup: RED sets up defenses, GRN is inside spawn. Damage is disabled.
  • Stage 1: No control points captured by GRN. ControlPoint1 can be captured by GRN.
  • Stage 2: CP1 captured by GRN. ControlPoint2 can be captured.

King Of The Hill

  • Stage 1: Match start, ControlPoint neutral.
  • Stage 2: CP captured by RED.
  • Stage 3: CP captured by GRN.

3 Control Points

  • Stage 1: ControlPoint2 neutral, only point capturable.
  • Stage 2: CP2 captured by RED; CP3 can be captured.
  • Stage 3: CP2 captured by GRN; CP1 can be captured.

Weapon Collisions

Items listed are blocked by the given part.

  • Hitscan (bullets, melee)
  • Projectiles (rockets, arrows, mad milk; excludes Annihilator’s projectiles)
  • Flames (flamethrower)
  • Grenades (grenade launcher)
  • Stickies (stickybombs)
  • Buildings
  • Players
  • Ragdolls

Geometry (CanCollide true, CanTouch true)

  • Hitscan
  • Projectiles
  • Flames
  • Grenades
  • Stickies (stick)
  • Buildings (can build)
  • Players
  • Ragdolls

Geometry (CanCollide false)

  • Hitscan
  • Projectiles (passes through; trajectory still collides)
  • Flames
  • Buildings (cannot build)

Geometry (CanCollide true, CanTouch false)

  • Hitscan
  • Projectiles
  • Flames
  • Grenades
  • Stickies (bounce)
  • Buildings (can build)
  • Players
  • Ragdolls

Ignore (CanCollide true)

  • Grenades
  • Stickies (stick)
  • Players
  • Ragdolls

Ignore (CanCollide false)

  • None

Ignore (CanCollide true, CanTouch false)

  • Grenades
  • Stickies (bounce)
  • Players
  • Ragdolls

Clips

  • Flames
  • Buildings (can build)
  • Players

KillBricks, Water, Cameras, JumpPads, Teleporters, SpawnZones

Same as Ignore with respective properties.

NoBuild

  • Buildings
  • Same as Ignore

Enemy Team’s Barrier

  • Grenades
  • Stickies (bounce)
  • Buildings (can build)
  • Players
  • Ragdolls
